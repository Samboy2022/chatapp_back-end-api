Messaging

Send, receive, and manage messages in chats.

Message Reactions

Add and manage emoji reactions to messages.

POST /chats/{chatId}/messages/{messageId}/react Requires Authentication

Add an emoji reaction to a message. If the user has already reacted with the same emoji, it will be removed (toggle behavior).

Path Parameters
Parameter Type Required Description
chatId integer Yes The unique identifier of the chat
messageId integer Yes The unique identifier of the message
Request Body
JSON Request 
{
  "emoji": "👍"
}
Response
Success Response (200) 
{
  "success": true,
  "message": "Reaction added successfully",
  "data": {
    "id": 1,
    "message_id": 115,
    "user_id": 1,
    "emoji": "👍",
    "created_at": "2025-07-14T19:45:30.000000Z"
  }
}

POST /groups/{groupId}/users Requires Authentication

Add a user to an existing group chat. Only group admins can add new members.

Request Body
JSON Request 
{
  "user_id": 5
}
Response
Success Response (200) 
{
  "success": true,
  "message": "User added to group successfully",
  "data": {
    "user": {
      "id": 5,
      "name": "Jane Smith",
      "avatar_url": null,
      "is_online": true
    },
    "role": "member",
    "joined_at": "2025-07-14T19:47:00.000000Z"
  }
}
DELETE /groups/{groupId}/users/{userId} Requires Authentication

Remove a user from a group chat. Only group admins can remove members, or users can remove themselves.

Path Parameters
Parameter Type Required Description
groupId integer Yes The unique identifier of the group
userId integer Yes The unique identifier of the user to remove
Response
Success Response (200) 
{
  "success": true,
  "message": "User removed from group successfully"
}
