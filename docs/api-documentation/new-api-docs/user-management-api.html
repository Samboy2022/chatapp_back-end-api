GET /settings/profile

Get authenticated user's profile information

Returns complete user profile data including account details and settings.

POST /settings/profile

Update user's profile information

Updateable Fields

name about email password
GET /settings/privacy

Get user's privacy settings

Privacy Options

last_seen_privacy: everyone, contacts, nobody
profile_photo_privacy: everyone, contacts, nobody
about_privacy: everyone, contacts, nobody
read_receipts_enabled: true, false
groups_privacy: everyone, contacts
status_privacy: everyone, contacts, close_friends
POST /settings/privacy

Update user's privacy settings

All fields are optional. Only provided fields will be updated.

GET /settings/media-settings

Get user's media and storage settings

Media Preferences

auto_download_photos: Auto-download photos
auto_download_videos: Auto-download videos
media_quality: high, medium, low
delete_media_after_days: Auto cleanup period
compress_images: Enable image compression
save_to_gallery: Auto-save to device gallery
GET /settings/notifications

Get user's notification settings

Notification Types

message_notifications: New messages
call_notifications: Incoming calls
group_notifications: Group activity
status_notifications: Status updates
notification_sound: Sound preference
vibrate: Vibration settings
POST /settings/notifications

Update user's notification settings

Customize notification preferences for different types of activities.

GET /settings/export-data

Export user's personal data

profile: Basic account information
privacy_settings: Privacy preferences
export_generated_at: Export timestamp
note: Export information
POST /settings/delete-account

Permanently delete user account

⚠️ Warning: This action cannot be undone. Requires password verification and confirmation text.

Required Fields

{
  "password": "current_password",
  "confirmation": "DELETE_MY_ACCOUNT"
}

💻 Code Examples

JavaScript (Profile Management)

// Get user profile
const getProfile = async () => {
  const response = await fetch('http://127.0.0.1:8000/api/settings/profile', {
    headers: { 'Authorization': `Bearer ${token}` }
  });
  return await response.json();
};

// Update profile
const updateProfile = async (data) => {
  const response = await fetch('http://127.0.0.1:8000/api/settings/profile', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${token}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify(data)
  });
  return await response.json();
};

// Update privacy settings
const updatePrivacy = async (settings) => {
  const response = await fetch('http://127.0.0.1:8000/api/settings/privacy', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${token}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify(settings)
  });
  return await response.json();
};

// Export user data
const exportData = async () => {
  const response = await fetch('http://127.0.0.1:8000/api/settings/export-data', {
    headers: { 'Authorization': `Bearer ${token}` }
  });
  return await response.json();
};

cURL Commands

# Get user profile
curl -X GET "http://127.0.0.1:8000/api/settings/profile" \
  -H "Authorization: Bearer TOKEN"

# Update profile
curl -X POST "http://127.0.0.1:8000/api/settings/profile" \
  -H "Authorization: Bearer TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Updated Name",
    "about": "Updated bio"
  }'

# Update privacy settings
curl -X POST "http://127.0.0.1:8000/api/settings/privacy" \
  -H "Authorization: Bearer TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "last_seen_privacy": "contacts",
    "read_receipts_enabled": true
  }'

# Get notification settings
curl -X GET "http://127.0.0.1:8000/api/settings/notifications" \
  -H "Authorization: Bearer TOKEN"

# Export user data
curl -X GET "http://127.0.0.1:8000/api/settings/export-data" \
  -H "Authorization: Bearer TOKEN"

# Delete account (⚠️ DANGER!)
curl -X POST "http://127.0.0.1:8000/api/settings/delete-account" \
  -H "Authorization: Bearer TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "password": "currentpassword",
    "confirmation": "DELETE_MY_ACCOUNT"
  }'

🔧 Integration Notes

  • Profile updates are immediately reflected across all sessions
  • Privacy settings take effect immediately for new interactions
  • Password changes require current password verification
  • Email changes may require verification in production
  • Data export includes all user data for GDPR compliance
